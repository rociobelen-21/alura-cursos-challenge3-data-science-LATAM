# Challenge3 Data Science LATAM
# Descripción del proyecto

Este proyecto forma parte del Challenge de Data Science de Alura LATAM, cuyo objetivo es aplicar técnicas de análisis de datos y machine learning para predecir la cancelación de clientes (churn).

La cancelación de clientes es uno de los principales problemas en empresas de telecomunicaciones. Mediante el análisis de datos y modelos predictivos se busca identificar los factores que influyen en la pérdida de clientes y generar estrategias para mejorar su retención.

# Objetivo

* Desarrollar un pipeline de análisis de datos que permita:

* Preparar los datos para el modelado.

* Analizar la relación entre variables y cancelación de clientes.

* Entrenar modelos de Machine Learning.

* Evaluar el rendimiento de los modelos.

* Identificar los factores que influyen en el churn.

# Tecnologías utilizadas

* Python

* Pandas

* NumPy

* Scikit-learn

* Matplotlib

* Google Colab

Estas herramientas se utilizaron para el procesamiento de datos, análisis exploratorio y entrenamiento de modelos de clasificación.

# Etapas del proyecto
# Extracción de datos

Se cargó el dataset tratado previamente en el desafío anterior.

# Preprocesamiento de datos

Se realizaron tareas de limpieza y preparación de datos:

* Tratamiento de valores faltantes

* Codificación de variables categóricas

* Normalización de variables

# Análisis exploratorio

Se analizaron las relaciones entre variables para identificar posibles factores asociados a la cancelación de clientes.

# Modelado predictivo

Se entrenaron modelos de clasificación para predecir el churn:

* Regresión Logística

* Random Forest

# Evaluación de modelos

Los modelos se evaluaron utilizando métricas como:

* Accuracy

* Precision

* Recall

* F1-score

# Resultados

El modelo Random Forest mostró un mejor desempeño para identificar clientes con mayor probabilidad de cancelar su servicio.

El análisis de importancia de variables permitió identificar factores clave relacionados con el churn.

# Principales factores que influyen en la cancelación

Entre las variables más relevantes se destacan:

* Tipo de contrato

* Tiempo de permanencia del cliente

* Cargos mensuales

* Tipo de servicio de internet

Estos factores pueden ayudar a la empresa a diseñar estrategias para reducir la cancelación de clientes.

# Conclusiones

El uso de modelos predictivos permite identificar clientes con riesgo de abandono antes de que cancelen su servicio.

A partir de estos resultados, las empresas pueden implementar estrategias de retención basadas en datos y mejorar la experiencia de sus clientes.
